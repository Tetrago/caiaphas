Detailed Description

Object responsible for configuring and constructing instances.

The builder object cannot be copied or moved, so the construction needs to be a one-shot call.

Instance mvtk::Instance::Builder::build ( )
inline

Constructs the instance using the configured options.

No error checking or input validation is applied up to this point. None of the past methods have checked the existence of the supplied extensions or layers, so any issues will arise here, upon creation. Thus, it is possible for this function to throw.

Returns
Constructed and usable Vulkan instance.

◆ debug()

Builder & mvtk::Instance::Builder::debug ( )
inlinenoexcept

Adds debugging features and configurations to the instance.

This primarily achieves four tasks:

  • The VK_EXT_DEBUG_UTILS_EXTENSION extension is added to the extensions list.
  • The VK_LAYER_KHRONOS_validation layer is added to the layers list.
  • A debug messenger is created that logs all Vulkan messages to "vulkan.log".
  • A secondary debug messenger with the same output location is applied to the instance creation and destruction.
